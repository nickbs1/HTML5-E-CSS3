Imagens

A tag <picture> envolve as diferentes fontes de imagem (<source>) e a imagem padrão (<img>).

<source> é usado para especificar diferentes fontes de imagem com base no tipo de arquivo e fornece a melhor opção para o navegador.
ex: <source media="(max-width:750px)" srcset="imagens/foto-p.png" type="image/png">
source media="(max-width:750px)"=isso especifica a condição de mídia que a fonte de imagem deve atender para ser considerada. Neste caso, a condição é definida para telas com uma largura máxima de 750 pixels.
srcset="imagens/foto-p.png"=Este atributo especifica o caminho para a imagem a ser utilizada se a condição de mídia especificada for atendida. Neste caso, "imagens/foto-p.png" é o caminho para a imagem em formato PNG.
type="image/png"=Este é o tipo da imagen.
